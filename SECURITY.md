# Security Policy

WrenchPro is currently in internal alpha / private beta preparation.

## Supported Versions

Only the latest approved internal build is supported during early testing.

## Reporting Security Issues

Do not post sensitive security details publicly.

For now, report security concerns directly to Brandon/Van through the approved private channel.

## Sensitive Data

WrenchPro may store customer, vehicle, job, payment, and business information locally. Treat this data as private business data.

Do not share databases, screenshots, logs, or customer records publicly unless they are sanitized.

## Current Security Priorities

- Confirm local database location.
- Confirm backup/restore procedure.
- Confirm update/reinstall preserves data.
- Avoid collecting centralized customer data until privacy/security plans are in place.
- Add clear support and vulnerability reporting process before public beta.
