# WrenchPro Feedback Inbox

Use this file to capture raw feedback from Brandon, his brother, testers, or future users.

## How To Use

Paste feedback here first, even if it is messy. Later, the Product Manager Agent can sort it into bugs, feature requests, workflow gaps, usability problems, or documentation needs.

## Feedback Categories

- Bug
- Feature Request
- Workflow Gap
- Usability / Confusion
- Performance
- Data / Backup Concern
- Documentation Need
- Pricing / Business Model
- Support Question
